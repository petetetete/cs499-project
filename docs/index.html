Introduction

How fragile is the modern web development ecosystem? How reliant are popular websites on external JavaScript libraries? These are a few of the common concerns present in modern day use of the web, especially for web developers in large companies. This is due to the fact that most websites rely on packages external to their codebase that are downloaded via a package manager such as npm.

Although, these package managers provide an overall positive service to the web development community, there are several different cases of package outages causing development issues. These outages also effect the users of these large business sites. This is why we will be collecting data about the dependencies of these JavaScript packages, and analyzing several network metrics to assess the potential scale of this issue, or to discover any possible strategies one could use to mitigate the effect of these problems.

For a more detailed description of our goal, see our initial project proposal.
